For specific queries, override staleTime:

─── 13. SEARCH DEBOUNCE ───
All search inputs across the app should debounce at 300ms before firing
the filter/query update. If not already using a debounce hook:

  function useDebounce<T>(value: T, delay: number): T {
    const [debounced, setDebounced] = useState(value)
    useEffect(() => {
      const t = setTimeout(() => setDebounced(value), delay)
      return () => clearTimeout(t)
    }, [value, delay])
    return debounced
  }

Apply this to all search inputs in: /resources, /tasks, /customers,
/opportunities, /projects, /timesheets/approval.
