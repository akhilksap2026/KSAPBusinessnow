// ── UX-04: HIERARCHICAL DATA FENCING ─────────────────────────────

6. Create server/utils/scopeFilter.ts. Export function buildUserScope(userId: number, db): Promise<{ userIds: number[], projectIds: number[] }>. It queries the reporting chain: find all users where reportsToId = userId (direct reports), then their direct reports recursively (up to 3 levels). Return the full set of userIds and their associated projectIds. Every API GET for /api/projects, /api/resources, /api/timesheets, /api/allocations must import and call this function to scope results. For Partner/Admin roles, bypass scoping and return all records.
