## 3. Role Auto-Population (Backend)

File: api-server/src/routes/timesheets.ts

On POST /api/timesheets:
Before insert:

1. Get allocation:
   SELECT allocationType FROM allocations
   WHERE resourceId=? AND projectId=? LIMIT 1

2. Get applicable rate card:
   SELECT id FROM rate_cards
   WHERE projectId=?
      OR (projectId IS NULL AND isTemplate=true)
   ORDER BY projectId DESC NULLS LAST
   LIMIT 1

3. Match role:
   SELECT sellRate, buyRate FROM rate_card_roles
   WHERE rateCardId=? AND roleName=resource.defaultRole

4. Set fields:
   IF match:
     billedRole = defaultRole
     sellRate = sellRate
     costRate = resource.costRate
   ELSE:
     billedRole = defaultRole
     sellRate = null
     costRate = resource.costRate

- This is NOT visible in UI
- Only stored for margin calculations
